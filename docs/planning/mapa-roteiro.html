Roteiro do Projeto

A espinha do projeto — 9 passos em 3 estágios. Estados dos nós vêm do board (campo Etapa + Status). Detalhe por epic: relatório de progresso.

Os 3 estágios

Definição passos 1–7
Descobrir e desenhar a solução — cada passo fecha via grilling.
Execução passo 8
Construir e entregar, por versão (v1 → v2 → v3).
Evolução passo 9
Manter e evoluir depois do lançamento.

Os 3 portões (gates)

G1 critério de avanço na Definição
Um passo só abre quando o anterior fecha (paralelos declarados são exceção).
G2 critério de avanço para a Execução
A Execução (passo 8) só começa quando toda a Definição (1–7) fecha — sinal: marco v0 a 100%.
G3 critério de avanço entre versões
Cada versão só começa quando a anterior passa no checklist de lançamento.
